Graphics Editor by Sorokin Dmitry

Требования:
    Библиотеки: tkinter
                PIL

Графический редкатор, способный редактировать изображения в любом формате.
Имеет поддержку:
    1) кисть и ластик разоного размера
    2) выбор цвета для кисти
    3) изменение размера хослта
    4) загрузка изображения
    5) работа со слоями
    6) редактирование гиф(разбиение на кадры)

run program.py
